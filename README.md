# NBox

**NBox** — локальное десктопное приложение для обработки мультимедийных и офисных файлов. Все операции выполняются офлайн, без отправки данных во внешние сервисы.

## [Скачать последнюю версию](https://disk.yandex.ru/d/IL_WyMCo-Z-9Vg)

## Возможности

- Сжатие видеофайлов (поддержка `.mp4`, `.mov`, `.avi`, `.mkv`, `.webm`)
- Конвертация PDF в:
  - `.txt` с извлечением текста и изображений
  - `.docx` (Word)
- Конвертация презентаций `.pptx` в `.docx` с поддержкой текста, заметок и изображений

## Технологии

- Python 3
- Tkinter (GUI)
- FFmpeg
- pdfplumber
- python-docx
- python-pptx
- PyMuPDF

## Преимущества

- Полностью офлайн
- Простое графическое управление
- Пакетная обработка файлов
- Низкие системные требования
